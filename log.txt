usage: tno.py [-h] [-s] [-r READ] [-n NEW] [-d DELETE] [-e EDIT]

A terminal based notes app

optional arguments:
  -h, --help            show this help message and exit
  -s, --show            Lists all notes
  -r READ, --read READ  Gets a note by name
  -n NEW, --new NEW     Creates new note
  -d DELETE, --delete DELETE
                        Delete note
  -e EDIT, --edit EDIT  Opens note in your default text editor
